# ソートアルゴリズム比較ツール

高等学校情報Ⅰの教材として開発された、複数のソートアルゴリズムを視覚的に比較できる教育用Webアプリケーションです。

## 🎯 プロジェクト概要

このアプリケーションは、生徒がアルゴリズムの動作を直感的に理解できるよう、5つの主要なソートアルゴリズムを視覚化して比較できる機能を提供します。

### 主な特徴

- ✅ **5つのソートアルゴリズムの実装**
  - クイックソート（Quick Sort）
  - マージソート（Merge Sort）
  - バブルソート（Bubble Sort）
  - 選択ソート（Selection Sort）
  - 挿入ソート（Insertion Sort）

- ✅ **アルゴリズム別の色分け視覚化**
  - **クイックソート**: ピボット要素を赤オレンジ色（#FF6B35）で明確に表示
  - **マージソート**: マージ中の要素を水色/シアン（#00BCD4）で表示
  - **選択ソート**: 現在の最小値を紫色（#AB47BC）で表示
  - **挿入ソート**: 挿入位置を探している要素を紫色（#AB47BC）、整列済み部分を薄緑色で表示
  - **バブルソート**: 比較中のペアを明るい黄色（#FFE082）、確定した要素を緑色で表示

- ✅ **視覚的な動作表示**
  - リアルタイムのバーチャート表示
  - 比較・交換・ソート済み要素の色分け
  - 各アルゴリズムに最適化された色の凡例

- ✅ **柔軟な実行制御**
  - 自動実行モード
  - 一時停止機能
  - ステップ実行（1ステップずつ確認）
  - 速度調整（10段階）

- ✅ **比較モード**
  - 複数のアルゴリズムを同時に実行
  - 比較回数・交換回数・実行時間の統計表示

- ✅ **カスタマイズ機能**
  - 配列サイズの調整（5〜50要素）
  - ランダム配列生成
  - カスタム配列入力

- ✅ **教育的な解説**
  - 各アルゴリズムの詳しい説明
  - 時間計算量・空間計算量の表記
  - アルゴリズムの特徴と適用場面

## 🚀 使い方

### 基本的な使用方法

1. **配列の設定**
   - スライダーで配列サイズを調整（5〜50）
   - 「ランダム生成」ボタンで新しい配列を生成
   - または「カスタム入力」でオリジナルの配列を入力

2. **アルゴリズムの選択**
   - 比較したいアルゴリズムにチェックを入れる
   - 複数選択可能

3. **実行設定**
   - 速度スライダーで実行速度を調整
   - 「比較モード」にチェックを入れると複数同時実行

4. **実行**
   - 「開始」ボタンでソート開始
   - 「一時停止」で中断
   - 「1ステップ」で段階的に確認
   - 「リセット」で初期状態に戻る

### 視覚化の見方（色の凡例）

#### 共通の色
- **紫色のバー**: 通常の未整列要素
- **明るい黄色のバー**: 比較中の要素（#FFE082）
- **赤色のバー**: 交換中の要素
- **緑色のバー**: ソート済み要素

#### アルゴリズム固有の色

**クイックソート:**
- **赤オレンジ色**: ピボット要素（基準値）- 明確に識別できる赤系の色（#FF6B35）
- 枠線付きで強調表示

**マージソート:**
- **水色/シアン**: マージ処理中の要素（#00BCD4）

**選択ソート:**
- **紫色**: 現在の最小値候補（#AB47BC）
- 枠線付きで強調表示

**挿入ソート:**
- **紫色**: 挿入位置を探している要素（#AB47BC）- パルスアニメーション付き
- **薄い緑色**: 整列済み部分

**バブルソート:**
- **明るい黄色**: 現在比較中のペア（#FFE082）
- **緑色**: 各パスで確定した要素

### 統計情報

各アルゴリズムの実行後、以下の情報が表示されます：
- **比較回数**: 要素を比較した回数
- **交換回数**: 要素を交換した回数
- **実行時間**: アルゴリズムの実行にかかった時間

## 📚 実装されているアルゴリズム

### 1. クイックソート (Quick Sort)
- **概要**: 分割統治法による高速ソート
- **視覚化の特徴**: ピボット要素が**赤オレンジ色（#FF6B35）**で明示され、枠線付きで強調表示される
- **時間計算量**: 平均 O(n log n)、最悪 O(n²)
- **空間計算量**: O(log n)
- **安定性**: 不安定
- **特徴**: 実用的に最も高速なアルゴリズムの一つ

### 2. マージソート (Merge Sort)
- **概要**: 配列を分割してマージする分割統治法
- **視覚化の特徴**: マージ中の要素が水色/シアンで表示される
- **時間計算量**: 常に O(n log n)
- **空間計算量**: O(n)
- **安定性**: 安定
- **特徴**: 安定したパフォーマンス、外部ソートに適している

### 3. バブルソート (Bubble Sort)
- **概要**: 隣接要素を比較して交換
- **視覚化の特徴**: 比較中のペアが明るい黄色、確定した要素が緑色
- **時間計算量**: 平均・最悪 O(n²)
- **空間計算量**: O(1)
- **安定性**: 安定
- **特徴**: 最もシンプルだが効率は悪い、教育用途に最適

### 4. 選択ソート (Selection Sort)
- **概要**: 最小値を選択して配置
- **視覚化の特徴**: 現在の最小値候補が**紫色（#AB47BC）**で枠線付きで表示される
- **時間計算量**: 常に O(n²)
- **空間計算量**: O(1)
- **安定性**: 不安定
- **特徴**: 交換回数が少ない

### 5. 挿入ソート (Insertion Sort)
- **概要**: 整列済み部分に挿入
- **視覚化の特徴**: 挿入する要素が**紫色（#AB47BC）**でパルスアニメーション、整列済み部分が薄緑色
- **時間計算量**: 平均・最悪 O(n²)、最良 O(n)
- **空間計算量**: O(1)
- **安定性**: 安定
- **特徴**: 小規模データや部分的にソート済みのデータに効率的

## 💡 教育的な活用方法

### 授業での活用例

1. **アルゴリズムの比較学習**
   - 複数のアルゴリズムを同時実行して効率を比較
   - 配列サイズを変えて計算量の違いを観察

2. **段階的な理解**
   - ステップ実行で各操作を詳しく確認
   - 色分けされた視覚化で処理の流れを把握
   - 凡例を参照して各色が何を意味するか学習

3. **実験と検証**
   - カスタム配列で特定のケースを試す
   - 最悪ケース・最良ケースの観察

4. **統計データの分析**
   - 比較回数や交換回数から効率性を評価
   - 計算量の理論と実測値の比較

## 🎨 色の凡例機能

各アルゴリズムの視覚化エリアには、そのアルゴリズムに特化した色の凡例が表示されます：

- **動的表示**: 選択したアルゴリズムに応じて自動的に最適な凡例を表示
- **視覚的理解**: 色と意味の対応を明確にすることで、処理の理解を促進
- **教育的価値**: 生徒が各アルゴリズムの特徴的な動作を色で識別できる

## 🎓 推奨される学習の流れ

1. **導入**: バブルソートから始めて基本を理解
2. **発展**: 選択ソート、挿入ソートで他の基本アルゴリズムを学習
3. **応用**: マージソート、クイックソートで分割統治法を理解
4. **比較**: 複数のアルゴリズムを同時実行して効率を比較
5. **実験**: カスタム配列でさまざまなケースを試す
6. **色の理解**: 各アルゴリズムの色の凡例を参照して、特徴的な動作を識別
